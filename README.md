# lua-timeago

Turn a timestamp into a friendly string of how long ago it was

## Usage

### 1. Install
Setup [lua-loader](https://github.com/wscherphof/lua-loader) and then just `npm install lua-timeago`

### 2. Require
```lua
local timeago = require("lua-timeago")
```
